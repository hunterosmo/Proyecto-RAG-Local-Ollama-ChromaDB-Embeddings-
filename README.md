# Proyecto-RAG-Local-Ollama-ChromaDB-Embeddings-
Sistema RAG local que combina Ollama y ChromaDB para consultar tus propios documentos (PDF, Word, Excel, PPTX, TXT, MD) sin usar la nube. Incluye menú interactivo, consola avanzada y router inteligente para chat, código y búsquedas precisas, todo ejecutado 100% offline.
